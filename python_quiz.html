1. What is the correct way to create a variable in Python?

A) var x = 5
B) x = 5
C) let x = 5
D) const x = 5
Correct Answer: B
In Python, you simply use the assignment operator (=) to create variables. No keywords like 'var', 'let', or 'const' are needed.

2. Which of the following is NOT a valid Python data type?

A) list
B) tuple
C) array
D) dictionary
Correct Answer: C
Python doesn't have a built-in 'array' type. It uses 'list' instead. While you can import arrays from the 'array' module, it's not a built-in data type.

3. What will be the output of: print(type([]))

A) <class 'array'>
B) <class 'list'>
C) <class 'tuple'>
D) <class 'set'>
Correct Answer: B
The empty square brackets [] create an empty list in Python, so type([]) returns <class 'list'>.

4. Which method is used to add an element to the end of a list?

A) append()
B) add()
C) insert()
D) push()
Correct Answer: A
The append() method adds an element to the end of a list. insert() adds at a specific position, while add() and push() are not list methods.

5. What is the correct syntax for a function definition in Python?

A) function myFunction():
B) def myFunction():
C) func myFunction():
D) define myFunction():
Correct Answer: B
In Python, functions are defined using the 'def' keyword followed by the function name and parentheses.

6. What will be the output of: print(3 * "Hello")

A) HelloHelloHello
B) 3Hello
C) Error
D) Hello 3
Correct Answer: A
Multiplying a string by an integer repeats the string that many times, so 3 * "Hello" results in "HelloHelloHello".

7. Which of the following is used for comments in Python?

A) //
B) /* */
C) #
D) --
Correct Answer: C
Python uses the hash symbol (#) for single-line comments. Multi-line comments can be written using triple quotes (''' or """).

8. What is the result of: len([1, 2, 3, 4, 5])

A) 4
B) 5
C) 6
D) Error
Correct Answer: B
The len() function returns the number of items in a sequence. The list [1, 2, 3, 4, 5] has 5 elements.

9. Which operator is used for exponentiation in Python?

A) ^
B) **
C) pow()
D) exp()
Correct Answer: B
The double asterisk (**) is the exponentiation operator in Python. For example, 2**3 equals 8.

10. What is the correct way to check if a key exists in a dictionary?

A) if key in dict:
B) if dict.hasKey(key):
C) if dict.contains(key):
D) if dict.keyExists(key):
Correct Answer: A
The 'in' operator is the Pythonic way to check if a key exists in a dictionary. The other methods don't exist in Python.
